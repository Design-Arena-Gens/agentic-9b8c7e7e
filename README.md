# Agentic ASP.NET-style Website (Next.js on Vercel)

This project delivers an ASP.NET-inspired website implemented using Next.js for effortless deployment on Vercel.

- Pages Router (`pages/`)
- TypeScript
- Serverless API route at `/api/hello`

## Scripts

- `npm run dev` - local development
- `npm run build` - production build
- `npm start` - run production server locally
